
Warehouse Management System


This is a backend Warehouse Management System built using Java and Spring Boot.
It helps manage warehouses, products, inventory, and orders through RESTful APIs.

Features

	•	Manage warehouses, products, inventory, and orders
	•	CRUD operations for all modules
	•	Inventory tracking across warehouses
	•	Order status handling using enums
	•	Clean layered architecture (Controller, Service, Entity)

Tech Stack

	•	Java
	•	Spring Boot
	•	Spring Web
	•	JDBC
	•	Maven

Project Structure

	•	Controller – Handles API requests
	•	Service – Contains business logic
	•	Entity – Represents database models
	•	Repository / DAO – Database interaction using JDBC

How to Run

	1.	Clone the repository
	2.	Open the project in an IDE (IntelliJ / Eclipse)
	3.	Configure database settings
	4.	Run the Spring Boot application

Purpose


This project is created for learning and demonstrating backend development concepts using Spring Boot and REST APIs.
